# 🐍 Python Engineering

## Overview
**The Goal:** To leverage Python's powerful ecosystem to build automated scripts, data engines, and full-stack applications.

* **For Non-Technical Readers:** Python is incredibly versatile. These projects range from an automated trading bot that analyzes the stock market to a personal expense tracker that saves your financial data.
* **For Technical Readers:** Utilizing Python for backend web development (Flask/Django), data analysis engines, and ORM database integrations (Prisma/PostgreSQL).

## Projects Included

1. **[Project-1: Flask Web App](./Project-1/)**
   * A full Python web application using the Flask framework.
2. **[Project-2: Bank Expense Tracker](./Project-2/)**
   * A terminal and backend-based application that allows users to track their finances, seamlessly integrated with a PostgreSQL relational database using Prisma ORM.
3. **[Project-3: Algorithmic Trading Engine](./Project-3/)**
   * A quantitative finance script that analyzes market data and executes algorithmic trading strategies.
