Windows WSL subsystem + Ubuntu version : Ubuntu-22.04

Command#1: wsl --install -d Ubuntu-22.04 //installing specific ubuntu version
Command#2: lsb_release -a // check for exact version

Phase 1: Set Up Your Profile
Create a Username: The terminal will ask you to create a "UNIX username". This can be anything (e.g., bhushan, robotics, etc.). Type it in and press Enter.

Create a Password: It will then ask for a password.

Crucial Linux Tip: When you type your password, nothing will show up on the screen (no stars or dots). This is normal Linux security! Just type your password and press Enter, then do it again to confirm.

Phase 3: Install the "Brain" (ROS 2 Humble)
Now we are going to install the ROS 2 middleware. Copy and paste these command blocks into your terminal one by one. Press Enter after each block and let it finish processing before pasting the next one.

(If it asks for a password, type the one you just created).

1. Set up your Locale (Ensures text encodes correctly):

Command#3: sudo apt update && sudo apt install locales -y
sudo locale-gen en_US en_US.UTF-8
sudo update-locale LC_ALL=en_US.UTF-8 LANG=en_US.UTF-8
export LANG=en_US.UTF-8

// 2. Enable the required Ubuntu repositories: 
Command#4: sudo apt install software-properties-common -y
sudo add-apt-repository universe -y 

3. Add the official ROS 2 security key (so your PC trusts the download):

Command#5: sudo apt update && sudo apt install curl -y
sudo curl -sSL https://raw.githubusercontent.com/ros/rosdistro/master/ros.key -o /usr/share/keyrings/ros-archive-keyring.gpg

//4. Add the ROS 2 download link to your system: 

Command#6: echo "deb [arch=$(dpkg --print-architecture) signed-by=/usr/share/keyrings/ros-archive-keyring.gpg] http://packages.ros.org/ros2/ubuntu $(. /etc/os-release && echo $UBUNTU_CODENAME) main" | sudo tee /etc/apt/sources.list.d/ros2.list > /dev/null

// 5. Install the ROS 2 Desktop Package! (Grab a cup of tea, this one is a large download. It includes RViz, math libraries, and the core ROS system)

Command#7: sudo apt update
sudo apt install ros-humble-desktop -y

//6. Source the environment: (This tells your terminal where ROS is installed so its commands work every time you open it).

Command#8: echo "source /opt/ros/humble/setup.bash" >> ~/.bashrc
source ~/.bashrc

//🌍 Phase 4: Install Gazebo Garden (The Physics Simulator)
1. Install necessary download tools:


Command#9:sudo apt-get update
sudo apt-get install wget curl lsb-release gnupg -y
 // 2. Add the official Gazebo security key:

Command#10: sudo wget https://packages.osrfoundation.org/gazebo.gpg -O /usr/share/keyrings/pkgs-osrf-archive-keyring.gpg

// 3. Add the Gazebo download repository to your system:

Command#11: echo "deb [arch=$(dpkg --print-architecture) signed-by=/usr/share/keyrings/pkgs-osrf-archive-keyring.gpg] http://packages.osrfoundation.org/gazebo/ubuntu-stable $(lsb_release -cs) main" | sudo tee /etc/apt/sources.list.d/gazebo-stable.list > /dev/null


//4. Install Gazebo Garden! (This is another large download because it contains 3D rendering engines and physics math libraries).

command#12: sudo apt-get update
sudo apt-get install gz-garden -y


🌉 Phase 5: Install the ROS-Gazebo Bridge
Since ROS 2 and Gazebo are two completely separate pieces of software, they need a "bridge" to pass messages to each other. Your document specifically mentions gazebo_ros_control, which is part of this bridge.

1. Install the bridge package for Humble and Garden:


command#13: sudo apt-get install ros-humble-ros-gzgarden -y


🧪 Phase 6: The Quick Test
Once everything is installed without errors, let's make sure the simulator actually opens! Because you are using WSL 2, Windows will automatically handle popping open the Linux graphical window.

Type this command into your terminal:


command#14: gz sim
