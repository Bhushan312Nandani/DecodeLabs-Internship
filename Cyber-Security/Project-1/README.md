# Network Security Scanner App

A web-based network monitoring and security scanning application combining a Python backend with an HTML/JS frontend.

## What It Does

- Scans network ports and services
- Displays security status via a browser interface
- Provides real-time monitoring feedback

## How to Run

```bash
pip install -r requirements.txt   # if applicable
python main.py
```

Then open `index.html` in your browser.

## Files

| File | Description |
|---|---|
| `main.py` | Python backend/scanner |
| `app.js` | JavaScript frontend logic |
| `index.html` | Browser UI |
| `style.css` | Styling |
