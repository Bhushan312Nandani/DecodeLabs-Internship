# Serverless Resume Uploader (AWS)

A serverless web app that lets users upload an HTML resume to AWS S3 via a pre-signed URL generated by Lambda. The resume is then publicly accessible via a permanent S3 link.

## Architecture

```
Browser → HTML Form → Lambda (Pre-signed URL) → S3 Bucket → Public URL
```

## What It Does

1. User selects an HTML file and enters admin password
2. Frontend requests a secure 5-minute upload URL from Lambda
3. File is uploaded directly to S3 (browser-to-S3)
4. A permanent public URL is returned for viewing the resume

## How to Set Up

1. Create an S3 bucket with public read access
2. Deploy the Lambda function from `serverless-resume-project/`
3. Set `LAMBDA_URL` in `index.html` to your function URL
4. Open `index.html` in a browser
